1.1.1.2.1.1.1.4.9. TraverseCallback¶

class molpher.swig_wrappers.core.TraverseCallback[source]¶

A simple callback that can process morphs in an exploration tree. When supplied to TraverseOper the morphs that belong to the currently traversed subtree are processed using this method.

Note

This is an abstract class. You will have to declare a derived class to use its features.

See also

TraverseOper

__call__(morph)[source]¶

An abstract method that should be overriden in derived classes. It should implement the morph processing.

Parameters:morph (MolpherMol) – currently processed morph
